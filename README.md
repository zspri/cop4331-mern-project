# cop4331-mern-project

This repository now includes a React Native starter app for the frontend mobile experience.

## Mobile App Starter

Path: `mobile-app/`

Features included:

- Workout tracking schema and starter screen
- Nutrition tracking schema and starter screen
- Daily dashboard with readiness score
- Coach tip logic based on recovery + activity

## Run Mobile App

```bash
cd mobile-app
npm install
npm run start
```

Open with Expo on Android emulator or physical device.

## Why This Is Useful For Your Team

Instead of shipping only a basic tracker, this starter already includes a "smart coach" concept (readiness + recommendation). It is still lightweight enough to finish quickly for class timelines.
